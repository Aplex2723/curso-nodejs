# Practica de Consola

Esta practica es una aplicacion de consola interactiva la cual tiene como funciones:

- Crear Tareas
- Listar las tareas
- Listar las tareas completadas
- Listar las tareas pendientes
- Completar tareas
- Borrar tareas
